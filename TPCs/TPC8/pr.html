Resume

  • Neste trabalho foi-nos pedido para criarmos uma Rest-API para obter os dados da ontologia cinema2020.

  • Para esse efeito foi criado um servidor utilizando o Express, e foram feitas as respetivas rotas para esse servidor.

  • As rotas são: /generos, /pais, /linguas, /atores, /atores/{:id}, /filmes, /filmes/{:id}, /personagens, /personagens/{:id}

  • O mesmo comunica com os endpoints do GraphDB e retorna os respetivos dados em JSON.
