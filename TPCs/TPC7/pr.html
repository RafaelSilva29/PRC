Resume

  • Neste trabalho era pretendido que fosse reunida uma boa quantidade de dados para povoar a Ontologia de Cinema. Estes dados foram obtidos através da API disponibilizada pela Movies Database.

  • Depois de conseguir reunir os dados necessários para preencher a ontologia, foi necessário escrever os triplos no formato Turtle para que pudessemos importar os dados no GraphDB.

  • Isto foi conseguido através da criação de uma script em Python que preparou os triplos em formato turtle para que estes fossem inseridos na ontologia.
