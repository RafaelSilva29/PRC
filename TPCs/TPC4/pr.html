Resume

  • Neste projeto foi pedido para criar uma aplicação muito simples que simulasse o GraphDB, fazendo a respetiva query em SPARQL e obter uma resposta do mesmo.

  • Para tal efeito foi utilizado o VueJS e o NodeJS, sendo que o primeiro faz de interface com o utilzador e o segundo faz de API para recolher os dados dos endpoins do GraphDB, fazendo assim pedidos aos mesmo para obter as respotas as querys efetuadas.

  • A aplicação permite escolher qual o repositório a utlizar bem como a query que se pretend fazer, os dados são dispostos em contéudo tabular, e também o possivel fazer download dos mesmos em formato JSON e CSV.
